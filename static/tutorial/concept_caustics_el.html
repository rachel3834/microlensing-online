Στην περίπτωση των βαρυτικών μικροφακών, οι καυστικές καμπύλες ανταποκρίνονται σε πολύ συγκεκριμένες θέσεις όπου η λύση της εξίσωσης της βαρυτικής εστίασης δίνει ώς αποτέλεσμα άπειρη μεγέθυνση. Για ένα σημειακό φακό (π.χ. ένα άστρο), η εξίσωση που μας δίνει τη μεγέθυνση είναι:

Μεγέθυνση =   A(u) =   u2 + 2
u (u2 + 4 )1/2

όπου το u συμβολίζει τις θέσεις της πηγής. Άν λύσετε την εξίσωση για u=0, τότε θα δείτε ότι η μεγέθυνση, Α, είναι άπειρη. Συνεπώς, για ένα σημειακό φακό, η καυστική καμπύλη είναι ένα μόνο σημείο, στο u=0.

Για ένα φακό που αποτελείται απο δύο σώματα (όπως ένα άστρο και ο πλανήτης του), η εξίσωση μεγέθυνσης είναι πιό πολύπλοκη, άλλα συνεχίζει να ισχύει ότι υπάρχουν λύσεις για τις οποίες η μεγέθυνση είναι άπειρη. Σε αυτή την περίπτωση οι καυστικές καμπύλες είναι πράγματι κλειστές καμπύλες, ή κάποιο σέτ απο κλειστές καμπύλες, και σε κάθε σημείο τους η μεγέθυνση της πηγής, σάν τις προσεγγίσει, τείνει στο άπειρο.

Λύνοντας λοιπόν την εξίσωση της βαρυτικής εστίασης, μπορούμε να προσδιορίσουμε ανα πάσα στιγμή τη μεγέθυνση της πηγής για οποιαδήποτε θέση της (μετά απο προβολή) σε σχέση με τη θέση του φακού. Έτσι μπορούμε να σχεδιάσουμε ένα χάρτη μεγέθυνσης που μας δείχνει την αναμενόμενη μεγέθυνση σε οποιδήποτε σημείο. Όσο μεγαλύτερη η μεγέθυνση τόσο λαμπότερη φαίνεται η πηγή. Η παρακάτω εικόνα δείχνει ένα χάρτη μεγέθυνσης, όπως επίσης και τις καυστικές καμπύλες, για ένα φακό που αποτελείται απο δύο σώματα.

Κάποιοι βασικοί κανόνες για τις καυστικές καμπύλες:

  1. Είναι κλειστές καμπύλες, οπότε μπορούμε να μιλάμε για 'μέσα' και για 'έξω'. (Για παράδειγμα, η περίμετρος ενός τριγώνου ή τετραγώνου είναι κλειστές καμπύλες, αλλά αναφερόμαστε μόνο στις γραμμές που περιέχουν το σχήμα και όχι στο σχήμα αυτό καθεαυτό.)

    PICTURE::FILENAME=caustic_inout.jpg::CAPTION=Caustic description::CREDIT=Jennifer Yee::NAME=Caustic description
  2. Οι καυστικές καμπύλες έχουν μηδενικό πλάτος. (Δηλαδή όπως και οι γραμμές στα μαθηματικά.)
  3. Στην καυστική καμπύλη, η μεγέθυνση της πηγής τείνει στο άπειρο.
  4. Η μεγέθυνση που ανταποκρίνεται σε σημεία μέσα στην καυστική καμπύλη είναι πάντα μεγαλύτερη απο τη μεγέθυνση απ'έξω.

  5. Τα άκρα της καυστικής καμπύλης ονομάζονται ακίδες. Τα καμπυλωτά τμήματα που τις ενώνουν ονομάζονται πτυχώσεις.
